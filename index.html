BitBulb

BitBulb is a coding idea generator, for those programmers who can't think of what type of project to make.

However BitBulb isn't just an idea generator, it also offers tutorials to a variety of programming languages.

So go check out some of the programming ideas and tutorials, and make sure to come back often, as BitBulb is always adding new ideas, tutorials, and updated tutorials.

Technology

Bitbulb is written in PHP, and uses Laravel(a PHP framework) to create a CRUD and a CMS, it uses SQL databases to store the tutorial posts information, it also uses Bootstrap to give it a more pleasant look to the site. On the backend we use Siteground(Website Hosting Service), which keeps the app live, and we use Namecheap for the domain name server
